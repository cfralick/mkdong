mkdong
======

A CLI utility to print dongs.    

__installation:__  
```bash
$ git clone https://github.com/cfralick/mkdong.git  
$ cd mkdong  
$ python setup.py install  
```    

__usage:__
```bash
$ python -m mkdong 5       
```

```python
>>> import mkdong     
>>> mkdong.dong(5)    
'( )/( )/////D'    
```
