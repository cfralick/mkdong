mkdong

A Python module for printing dongs.
This module borrows heavily from https://github.com/jathanism/mkdong.git

installation: 

$ git clone https://github.com/cfralick/mkdong.git  
$ cd mkdong  
$ python setup.py install

usage:

$ python -m mkdong 5       
>>> import mkdong     
>>> mkdong.dong(5)    
'( )/( )=====D'

help:

$ python -m mkdong -h
usage: mkdong [-h] [-w] [-c] [-v] l

Prints a dong.

positional arguments:
  l              the desired dong length

optional arguments:
  -h, --help     show this help message and exit
  -w, --wide     make a wide, thick dong
  -c, --climax   makes the dong climax
  -v, --version  print version and exit
